# ffmpeg的编译
```
git clone https://github.com/nldzsz/ffmpeg-build-scripts.git
cd ffmpeg-build-scripts
```
1、编译iOS平台的库
```
./compile-ffmpeg-ios.sh all
```
2、编译android平台的库
```
./compile-ffmpeg-android.sh all
```
3、编译mac平台的库
```
./compile-ffmpeg-mac.sh all
```
